## プロジェクト: yamato-portfolio

このレポートは、yamato-portfolioプロジェクトの概要、構造、使用技術、および開始方法について説明します。

### プロジェクト概要

このプロジェクトは、Next.jsで構築されたYamato個人のポートフォリオウェブサイトです。職務経歴、技術スキルを紹介し、ブログ機能を備えています。

### 使用技術

- **フレームワーク:** Next.js
- **言語:** TypeScript
- **UIライブラリ:** React
- **スタイリング:** CSS Modules
- **ブログコンテンツ:** Markdownを使用し、フロントマターには`gray-matter`、レンダリングには`remark`/`remark-html`を利用しています。
- **お問い合わせフォーム:** `nodemailer`を使用してメールを送信します。

### プロジェクト構造

プロジェクトの構成は以下の通りです:

- **`src/app`**: Next.jsのApp Routerを使用したコアアプリケーションが含まれます。メインレイアウト(`layout.tsx`)とホームページ(`page.tsx`)が含まれます。
- **`src/components`**: ポートフォリオの各セクション用の再利用可能なReactコンポーネントが格納されています:
  - `Header`: メインのナビゲーションヘッダー
  - `profile`: プロフィールセクション
  - `career`: 経歴セクション
  - `TechStack`: 技術スタックセクション
  - `ContactForm`: お問い合わせフォームコンポーネント
- **`src/pages/blog`**: Next.jsのPages Routerを使用してブログ機能を実装しています。
  - `index.tsx`: すべての投稿を一覧表示するブログのインデックスページです。
  - `[slug].tsx`: 個々のブログ投稿を表示するための動的ルートです。
- **`content/blog`**: ブログの投稿をMarkdownファイルとして保存します。各ファイルの内容はブログページで動的にレンダリングされます。
- **`public`**: 画像やロゴなどの静的アセットが含まれます。
- **`src/lib`**: お問い合わせフォームからのメール送信を処理する`sendMail.ts`などのユーティリティ関数が含まれます。

### 開始方法

このプロジェクトをローカルで実行するには、次の手順に従ってください:

1.  **依存関係のインストール:**

    ```bash
    npm install
    ```

2.  **開発サーバーの実行:**
    ```bash
    npm run dev
    ```
    これにより、Turbopackを使用した開発サーバーが起動します。ブラウザで[http://localhost:3000](http://localhost:3000)を開いて確認してください。

### 利用可能なスクリプト

- **`npm run dev`**: 開発サーバーを起動します。
- **`npm run build`**: 本番用にアプリケーションをビルドします。
- **`npm run start`**: 本番サーバーを起動します。
- **`npm run lint`**: Linterを実行してコードの品質をチェックします。
